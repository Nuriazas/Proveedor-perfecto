📖 Documentación de la Aplicación

🚀 Descripción General

SkillAgora es una plataforma de gestión de tareas y freelancing que permite organizar proyectos, gestionar tareas personalizadas y hacer seguimiento del progreso de desarrollo.

🔗 Endpoints y API

GET
/api/tasks
Obtiene todas las tareas del proyecto
POST
/api/tasks
Crea una nueva tarea personalizada
DELETE
/api/tasks/:id
Elimina una tarea específica

⚙️ Funcionalidades

📊
Dashboard Interactivo
Visualización en tiempo real del progreso de tareas por categorías
➕
Tareas Personalizadas
Crear, editar y eliminar tareas propias con subtareas y etiquetas
🔍
Sistema de Filtros
Filtrar tareas por categoría, estado y tiempo estimado
📈
Estadísticas
Métricas detalladas y estimaciones de finalización

🛠️ Tecnologías

Frontend: HTML5, CSS3, JavaScript ES6+
Almacenamiento: LocalStorage para persistencia de datos
Arquitectura: Módulos ES6, Sistema de componentes reutilizables
UI/UX: Diseño responsivo, temas claro/oscuro
